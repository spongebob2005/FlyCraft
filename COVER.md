# FlyCraft 1.0 Cover Banner

This file represents the visual cover/banner for FlyCraft 1.0 mod.

For a professional GitHub display, use one of these implementations:

## Option 1: SVG Banner (Recommended)

The banner is designed to be visually similar to pattern-craft with:
- Modern gradient background (cyan to purple)
- Bold "FlyCraft 1.0" typography
- Animated flight theme elements
- Elytra silhouette
- Particle effects concept
